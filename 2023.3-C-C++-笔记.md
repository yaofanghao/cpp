# C / C++
* 左值和右值
  * 左值lvalue 指向内存位置，可以被修改
  * 右值rvalue 指向存储在内存的数值
  
* 短路求值 
  
* static
  * 持久化 隐藏 默认初始化为0
  * 可以在函数调用间保持局部变量值
  * 可以定义静态成员和静态成员函数
  
* extern
  * 可以用于另一个文件中声明全局变量或函数 
  
* const
  * 不可改变值
  
* extern const
  * 在另一文件中引用const常量
  
* extern"C"
  * 使编译器对这部分按照C语言进行编译
  
* struct 和 union
  * union有数据对齐
  
* 静态库 动态库
  * 静态库是先吃撑了，防止之后饿
  * 动态库是把粮食带着，等饿了再吃
  
* 声明和定义
  * 声明不分配空间
  * 声明可以有多个，定义只有一个
  
* inline 内联函数
  * 用于频繁被调用的函数中
  * 以空间换时间
  
* namespace
  * 区分不同库中的相同函数、变量等
