 ```python
def flattern(A):
    result = []
    sta = list()
    for ele in reversed(A):
        sta.append(ele)
    while len(sta) > 0:
        ele = sta[-1]
        sta.pop()
        if isinstance(ele, int):
            result.append(ele)
        else:
            for sub_ele in reversed(ele):
                sta.append(sub_ele)
    return result
```
